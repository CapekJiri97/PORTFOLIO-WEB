Loan Eligibility Predictor

A comprehensive analysis of loan repayment ability using 10,000+ records. From raw data to a deployed Logistic Regression model.

Try Live App

1. Data Preparation & Cleaning

The project utilizes a synthetic dataset designed for analytical case studies, simulating real-world credit risk scenarios. It includes key attributes such as applicant's identification number, age, gender, marital status, and income group. I removed less relevant columns like rate and cost_type to simplify the analysis and improve model accuracy.

2. Preprocessing & Encoding

Transforming categorical variables into numerical ones using One Hot Encoding ensured that the model works with properly formatted data.

df_loan_dummies = [
  pd.get_dummies(df_loan["MarStat"], prefix="MarStat"),
  pd.get_dummies(df_loan["Housing"], prefix="Housing"),
  pd.get_dummies(df_loan["Inc_Gr"], prefix="Inc_Gr")
]

3. Train / Test Split

Splitting the dataset into training (80%) and testing (20%) sets ensured that the model could be accurately evaluated on independent data.

from sklearn.model_selection import train_test_split

df_train, df_test, y_train, y_test = train_test_split(
  df_loan.drop("Default_flg", axis=1),
  df_loan["Default_flg"],
  test_size=0.2,
  random_state=1337
)

4. Model Evaluation

Utilizing metrics such as Confusion Matrix, ROC AUC score, and K-S test on the test set allowed for a comprehensive evaluation of predictive power.
